# comp637-2026spring-finalproject

 In this repo, we only have two files. One is the paper we submit to gradescope, the other is the properties we wrote for Hack@DAC21's SoC.

 We have 21 properties adopted from the AKER paper, and 17 of them are instantiated using Hack@DAC21's Interconnect.

 Every property has 8 attributes （the first 5 are from the AKER paper's repo):
 - SP ID: 01-21
 - SP Type: IFT/Trace
 - Verification Level: IP/Firmware/System
 - Requirement: Describing property in natural language
 - SP Specification Template
 - Adaptation to Hack@DAC21: Describing our reasoning and the approximated property for Hack@DAC21
 - Sample Module: Controller/Peripheral module
 - Verfication Result: Violated/Satisfied, vacuity
